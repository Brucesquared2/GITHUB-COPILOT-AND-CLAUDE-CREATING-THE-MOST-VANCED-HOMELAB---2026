# AI Agent Cookbook: Terminal/TUI Shortcuts

| Agent | Command/Shortcut | Example Use Case | Description |
|-------|-----------------|------------------|-------------|
| Ollama | make run-ollama | Q&A, Summaries   | Run LLM prompt: `make run-ollama PROMPT="..."`
| HuggingFace | make run-hf | Sentiment/Classify| Run a pretrained model pipeline
| Opencode | make run-opencode | Coding agent | Use opencode for code reasoning/autocomplete
| OpenCLAW | make run-openclaw | RL/NLP Control | Send to OpenCLAW agent, retrieve insight |
| All | make tui | Interactive multi-agent TUI | Omni-dashboard for model/agent choice |
